# SafariMate Frontend

Modern, mobile-first frontend for SafariMate - Muslim Travel Companion.

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Zustand (State Management)
- Axios (API Client)

## Design System

- **Primary Color**: Deep emerald green (#0e4f45)
- **Accent Color**: Bright green (#24c186)
- **Gold Accent**: #e5c55b
- **Mobile-First**: Optimized for 390px width base
- **Islamic-Inspired**: Soft geometric patterns and elegant typography

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
